Extension Introduction

Hello friends, if you are reading this text, please continue reading because today we have brought an extension that can be very useful for our community 🤝.

With this extension, you can translate any text into any language, and that too very quickly 🚀.

Now, let's see how to use it. You don't need to do anything, just run it, and you will see three tabs above:

  • To – Contains all the available languages.
  • Recent – Shows the languages you have used recently.
  • About – Contains information about the extension 📚.

Whatever language you use will be saved in the Recent tab, so you can easily use it again later 📝.

Now, focus on the text you want to translate, apply this extension to it, and select the language you want to translate it into 📊.

For example, if your text is in Urdu and you want to translate it into English, simply choose English from the extension 🇬🇧.

If a text box is available, the extension will translate the text and immediately paste the translated version into the text box. However, it will first clear the contents of the text box. Therefore, it is recommended to keep the text box empty before running the extension 📝.

If you are wondering whether the existing text in the text box can be preserved, the answer is no. Otherwise, the translated text would simply be added on top of the existing text, which is why we did not consider that feature necessary 🙅‍♂️.

Now, without further ado, let's wrap things up and end this post here 📣.
